# 数据类型

| 类型描述 |  |
| --- | --- |
| 数字型（Number） | JavaScript 中的双精度浮点型格式 |
| 字符串型（String） | 双引号包裹的 Unicode 字符和反斜杠转义字符 |
| 布尔型（Boolean） | true 或 false |
| 数组（Array） | 有序的值序列 |
| 值（Value） | 可以是字符串，数字，true 或 false，null 等等 |
| 对象（Object） | 无序的键:值对集合 |
| 空格（Whitespace） | 可用于任意符号对之间 |
| null | 空 |
